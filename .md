```mermaid
graph TD
    %% Main entry
    Start([User Visits Site]) --> Home[Home Page /]

    %% Unauthenticated flows
    Home -->|Click Sign In| Login[Login Page /login]
    Home -->|Click Register| Register[Register Page /register]
    Home -->|Click Browse Campgrounds| CampgroundsList[Campgrounds List /campgrounds]
    Home -->|Click Book Now| BookingForm[Booking Form /booking]

    %% Authentication
    Login -->|Submit credentials| Auth{NextAuth}
    Register -->|Submit form| Auth
    Auth -->|Success| CampgroundsList
    Auth -->|Failure| Login

    %% Campgrounds browsing
    CampgroundsList -->|Click campground| CampgroundDetail[Campground Detail /campgrounds/:cid]
    CampgroundDetail -->|Click Book| BookingForm

    %% Booking flow
    BookingForm -->|Submit| CreateBooking[Create Booking API]
    CreateBooking -->|Success| MyBookings[My Bookings /mybooking]
    CreateBooking -->|Error| BookingForm

    %% Manage bookings
    MyBookings -->|Edit| EditBooking[Edit Booking /booking/edit/:bid]
    MyBookings -->|Delete| DeleteBooking[Delete Booking API]
    DeleteBooking -->|Success| MyBookings
    EditBooking -->|Update| UpdateBooking[Update Booking API]
    UpdateBooking -->|Success| MyBookings

    %% Admin
    MyBookings -->|Admin view| AdminAllBookings[Admin All Bookings]
    AdminAllBookings -->|Edit Delete| EditBooking
    AdminAllBookings -->|Delete| DeleteBooking

    %% Sign out
    TopMenu -->|Sign Out| SignOutConfirm[Sign Out Confirm /signout]
    SignOutConfirm -->|Confirm| SignOut[Logout] --> Home
    SignOutConfirm -->|Cancel| PreviousPage[Return]

    %% Navigation
    subgraph Navigation
        TopMenu[TopMenu Component]
        TopMenu --> Home
        TopMenu --> Login
        TopMenu --> Register
        TopMenu --> CampgroundsList
        TopMenu --> BookingForm
        TopMenu --> MyBookings
    end

    %% UI Components
    subgraph UI
        Banner[Banner]
        Card[Card]
        DateReserve[Date Picker]
    end

    CampgroundsList --> Card
    Home --> Banner
    BookingForm --> DateReserve
    EditBooking --> DateReserve
